## Evidence hierarchy

1. **Strongest:** immutable source, deterministic artifact, exact replay, semantic validator, focused negative tests, commit-scoped CI, independent review, measured run.
2. **Moderate:** internally reviewed contract or authored fixture with explicit limitations.
3. **Weak:** design intention, planned evaluation, selected example, or inferred educational effect.
4. **Prohibited inflation:** treating CI success as learner benefit; treating browser parsing as native execution; treating progress as mastery; treating generated-media attractiveness as learning; treating selected agent anecdotes as longitudinal effectiveness.

## Claim-update rule

A claim may move upward only when the backlog item names the required evidence, the evidence is retained at an immutable revision, validators pass, limitations are recorded, and any required independent or ethics review is complete.
