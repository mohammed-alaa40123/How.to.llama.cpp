## Evidence inflation rules

Passing CI is not learner benefit. Browser parsing is not native execution. Model-free launch is not inference. Authored replay is not runtime capture. Progress is not mastery. Attractive generated media is not educational evidence. A selected agent sample is not a longitudinal result.
