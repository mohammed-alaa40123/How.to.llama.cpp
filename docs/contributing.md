# Contributing

See the repository-level [CONTRIBUTING.md](https://github.com/mohammed-alaa40123/How.to.llama.cpp/blob/main/CONTRIBUTING.md).

## Good first contributions

- Verify one call-chain edge and add a pinned source link.
- Explain one data structure with ownership and lifetime.
- Convert an existing conceptual figure into editable Mermaid.
- Review one PR/discussion and record whether it is merged, reverted, superseded, or still proposed.
- Add a backend capability note with official vendor documentation.
- Reproduce one runtime observation and add commands, environment, and raw outputs.
